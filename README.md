# Waris App
An application that promotes the preservation of Indonesian culture by digitizing the tradition of **Arisan** in Indonesia. It was featured in the Pijar Camp Hackathon by Telkom Indonesia in 2023 and secured a Top 5 position. The code in this repository is only a partial display and is solely used as a preview of the Waris App. This repository is intended to showcase a portfolio for the GDGoC UGM application in 2025.

## Tech Stack
* **Framework** : Flutter
* **Language** : Dart
* **State Management** : GetX
* **UI Toolkit** : Custom components, Google fonts, Flaticon
* **Tools Pendukung** : VS Code & Android Studio
